# 数据脚本目录

本目录包含菜谱数据分析系统的数据采集、清洗、导入和模拟脚本。

## 目录结构

```
data-scripts/
├── spiders/           # 爬虫脚本 - 从美食网站爬取菜谱数据
├── cleaning/          # 数据清洗脚本 - 清洗和标准化爬取的数据
├── importing/         # 数据导入脚本 - 将清洗后的数据导入数据库
└── simulation/        # 用户行为模拟脚本 - 模拟用户行为数据
```

## 使用说明

### 1. 爬取数据 (spiders/)
从下厨房、美食杰等网站爬取菜谱数据。

### 2. 清洗数据 (cleaning/)
对爬取的数据进行清洗、去重、标准化处理。

### 3. 导入数据 (importing/)
将清洗后的数据批量导入 MySQL 数据库。

### 4. 模拟行为 (simulation/)
生成模拟用户行为数据，用于行为分析功能。

## 执行顺序

1. 爬取数据 → 2. 清洗数据 → 3. 导入数据 → 4. 模拟行为

## 注意事项

- 爬虫请遵守目标网站的 robots.txt 和使用条款
- 请设置合理的爬取间隔，避免对目标服务器造成压力
- 确保数据库连接配置正确后再执行导入脚本
- 模拟数据仅用于开发和测试，不代表真实用户行为
