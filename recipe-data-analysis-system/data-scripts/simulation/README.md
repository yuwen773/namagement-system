# 用户行为模拟脚本

本目录包含模拟用户行为数据的脚本，用于生成用户行为分析所需的数据。

## 模拟行为类型

1. **登录行为** - 用户登录记录
2. **搜索行为** - 搜索菜谱关键词
3. **浏览行为** - 浏览菜谱详情
4. **收藏行为** - 收藏/取消收藏菜谱
5. **页面访问** - 各页面访问路径

## 脚本说明

### simulate_users.py
生成模拟用户账号（100+ 个）

### simulate_behaviors.py
生成用户行为数据：
- 每个用户生成 20-100 条行为记录
- 行为时间分布在过去 30 天内
- 模拟真实用户行为模式

### simulate_page_visits.py
模拟页面访问路径和停留时间

## 使用方法

```bash
# 1. 先生成模拟用户
python simulate_users.py

# 2. 生成行为数据
python simulate_behaviors.py

# 3. 生成页面访问数据
python simulate_page_visits.py
```

## 行为分布规则

- 登录：每个用户 2-10 次/月
- 搜索：每次登录 1-5 次
- 浏览：每次登录 5-20 次
- 收藏：浏览量的 5%-15%
- 活跃时间：早 8 点 - 晚 10 点

## 注意事项

- 模拟数据仅用于开发和测试
- 数据分布尽量接近真实情况
- 收藏量不应超过点击量
- 行为时间应符合逻辑顺序
