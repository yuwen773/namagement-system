**URL 格式**：

- 分类页：`https://www.xiachufang.com/category/`

**关键数据字段与选择器映射**：

| 字段 | CSS选择器 | 数据库映射 |
|:-----|:----------|:-----------|
| 菜谱名称 | `h1.page-title` | `recipes.name` |
| 成品图片 | `.recipe-cover img` | `recipes.image_url` |
| 用料列表 | `.recipe-ingredients` | `recipe_ingredients` |
| 制作步骤 | `.recipe-steps` | `recipes.steps` |
| 小贴士 | `.recipe-tip` | 额外提示 |
